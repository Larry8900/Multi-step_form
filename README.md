# Multi-step_form

A simple multi-step form web app built with HTML, CSS, and vanilla JavaScript. It guides users through multiple steps, validates inputs, and shows a final review before submission.

Live Site: https://multi-step-form-675w.vercel.app/

Github Repo: https://github.com/Larry8900/Multi-step_form.git

Features

🧭 Multi-step navigation (Next / Previous)
✅ Input validation before moving to the next step
📊 Progress indicator showing current step
📝 Collects user details across multiple sections
🔎 Review screen before submission
🎉 Success message after completing the form

Project Structure
├── index.html      — Form layout and steps structure  
├── styles.css      — All styling and transitions  
└── script.js       — Step navigation and validation logic  
Getting Started

Clone or download the project folder
Open index.html in your browser — no build tools needed

Built With

HTML5 / CSS3
Vanilla JavaScript (no frameworks)
Google Fonts (optional)
